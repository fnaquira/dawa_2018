# React Quiz
A JavaScript slide quiz built with React.

Getting Started
---------------

```shell
$ git clone https://github.com/fnaquira/dawa_2018.git
$ cd dawa_2018
$ mongorestore -d dawa_2018 bd/dawa_2018
$ cd backend
$ npm install
$ npm start
```

App Structure
-------------

```
QuizApp
└── Quiz
    ├── QuestionList
    |   └── Question
    |       └── Answer

```
